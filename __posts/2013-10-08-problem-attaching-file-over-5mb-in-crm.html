Hi,

I was trying to attach a file over 5Mb and the following error message appeared:


After some research from the error message, i was redirected to the codeplex website, and instantly i remembered, from here i'm using a solution for the auto number, is possible that the error comes from there?

Yes, it is.

So, from this article, i saw that the solution is deactivate the step that gives the auto number for the entity, and register only the step for the entities that we need.

I have to say that i'm happy with the auto number solution from the codeplex and the only problem that i had or i'm having is this one. I can say that using this solution saved time, we didn't have to develop a custom solution for this purpose.
