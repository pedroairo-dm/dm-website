As described on the post "CRM 2015 - Web Proxy - New service proxy classes for the Organization Service", new messages can be found on the Deployment Web Service that can help us on specific requirements, and it is never bad if we now at least the existence of them. Below are few new messages. 


Request class name

Description

AddCrmDatabaseRequest

Adds a CRM database to your deployment from a database 
backup or from a detached
database.

DeleteOrganizationRequest

Deletes a CRM organization.

RetrieveMonitoringResultsRequest

Retrieves results from the 
monitoring service operation
 tests for your CRM deployment.

RetrieveUserMappingRequest

Retrieves the mapping information of
existing users in a CRM organization.
